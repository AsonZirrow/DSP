# Engineering Progression

Rather than representing isolated applications, these repositories explore complementary aspects of designing modular and explainable software systems.

---

## Routing Decision Engine

**Engineering Focus**

Structuring computation and decision workflows.

This repository demonstrates how computation, configuration, domain models, and orchestration can be organized into independent architectural layers that form a reusable foundation for configurable decision systems.

↓

## Metadata Extraction Toolkit

**Engineering Focus**

Extracting and organizing information.

This repository demonstrates how raw files can be transformed into structured, analyzable information through modular analysis components, plugin-oriented architecture, automated validation, and consistent reporting.

↓

## Rule-Based Decision Engine

**Engineering Focus**

Evaluating information through explicit decision logic.

This repository demonstrates deterministic decision-making using configurable rules, conflict resolution, and explainable execution, where every decision follows an observable evaluation process.

---

Together, these repositories explore complementary engineering techniques for building software systems that are modular, maintainable, explainable, and extensible.
